DOSSIER — The Instrument Press. Paper-and-ink editorial with one hot proof-mark accent. The type system IS the brand: oversized tabular numerals, a numbered editorial index for a home, split-flap hero figures, bidi-isolated data chips, and a calligraphic Nastaliq masthead as the RTL signature.

Production fonts (not loaded here; prototype uses system fallbacks): Latin display & UI — Roboto Flex (wght 100–1000 + wdth 25–151, for the variable-urgency move) or Anybody. Instrument numerals — Martian Mono / IBM Plex Mono (deliberately not Space Mono). RTL body — Vazirmatn (weight-only urgency). Nastaliq signature word — Gulzar / Noto Nastaliq Urdu (Persian).
