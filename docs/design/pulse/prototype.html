PULSE — A vitals chart for your car. Car-as-living-body: a single breathing vitals pulse-line home with no visible list; emotional temperature scoped to the aching card with a capped ambient halo (max = saffron), paid off by "the exhale" on every relieving action. Rooms navigation (Cockpit / Garage / Pit-lane) with a persistent keypad quick-add.

Production type: Latin — Hanken Grotesk 400/600 with tabular numerals. Persian / Arabic / Sorani Kurdish — Vazirmatn variable (distinct Persian ۰۱۲۳ vs Eastern-Arabic ٠١٢٣). Persian masthead — Noto Nastaliq Urdu (or Gulzar), true Nastaliq; Arabic milestones — Aref Ruqaa. This mockup substitutes system stacks (CSP blocks webfonts); numeric fields use a tabular-numeral monospace fallback. Haptics (the accessible "exhale" channel) are native-only — a visual settle stands in here. Motion respects prefers-reduced-motion (breathing falls back to static). Emotional tints stay ambient; text/controls always meet WCAG contrast.
